This site is hand-crafted, hand-failed

As I built this site, I failed it time and time again. I checked my code, tested my code, tested my design. I changed things that weren't working along the way. I'm still fixing bugs and implementing new features as I go. It's kind of the only way to keep up in a webby web world.
